# Мультиклассовый Классификатор для изображений высокого разрешений (от 512x512 пикселей) #
Специально разработан, чтобы отличать разные классы по мелким различиям.
Использует EfficientNetB0 в качестве основы для модели.


Как использовать:
train.py – для тренировки и схоранения модели, для этого создайте папку dataset и в ней два каталога train и val в каждом из которых будут папки классов.

Пример:

```

dataset/
	train/
		class1/
		class2/
		...
	val/
		class1/
		class2/
		...

```

Модель автоматически найдёт все классы и обучиться на них. Обязательно нужно равное количество изображений для тренировки и валидации, чтобы классфикатор правильно обучился.


predict.py – для проверки модели пользователем, имеет интерфейс для удобной загрузки изображения или папки с изображениями. Для его работы обязательно должна присутствовать модель после работы train.py (multiclass\_model.keras) и файл со сохранёнными названиями классов (class\_names.json).
